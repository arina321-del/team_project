# team_project
Итоговый командный проект
## Описание проекта

Данный проект — консольное Java-приложение для работы со списком пользователей.
Приложение позволяет:
- загружать пользователей из разных источников (файл, ручной ввод, генерация);
- валидировать данные пользователей;
- сортировать пользователей по имени, email и паролю;
- выполнять параллельный подсчёт пользователей;
- сохранять результаты в файл.

---

## Используемые технологии

- Java 17+
- Maven
- Git / GitHub
---
## Структура проекта

Основные классы

- Main (содержит метод main, запускает работу приложения, вызывает ввод данных, сортировку, подсчёт и сохранение результатов)

- User (Хранит данные пользователя: имя, email, пароль, создаётся через User.Builder, используется во всём проекте)

- UserValidator (Валидация данных пользователя, проверяет: корректность имени, корректность email, надёжность пароля.

Источники данных

- InputSource и его реализации.


Многопоточность

- ThreadCounter (Использует пул потоков, параллельно считает количество совпадений пользователей)

Работа с файлами

- FileHandler (Записывает результаты работы программы в файл output_users.txt)

Тесты

- Ручные тесты, запускаемые отдельно.

---
Запуск консольного приложения

- Запуск осуществляется через класс Main и метод main.
